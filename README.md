# 2015 US Flight Delays Analysis

**Tools:** Python (pandas, matplotlib), SQL (sqlite3), Tableau  
**Dataset:** 2015 US Domestic Flights — 5.8M flights from Kaggle

## Project Goal
Analyze 2015 US domestic flight data to identify what causes delays and which airlines and airports perform the worst.

## Key Findings
1. **Spirit and Frontier** have the worst on-time performance (avg 14.5 and 12.5 min late)
2. **June** is the worst month to fly; September and October are the best
3. **Late aircraft cascades** are the #1 delay cause — one late flight triggers delays all day
4. **Chicago O'Hare** is the most delayed major airport
5. **Early morning flights** arrive early on average; evening flights average 10+ min late
6. **American Eagle** has the highest cancellation rate (5.1%)
